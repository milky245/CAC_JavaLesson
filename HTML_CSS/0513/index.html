HTML 语言编程学习主页

这里整理了常见 HTML 学习内容：文本格式、装饰方法、排版布局、图片音频等媒体插入、链接跳转以及高级操作。每个模块都包含代码样例和效果展示，适合做入门参考页。

文本格式 装饰方法 排版方法 媒体插入 链接跳转 高级操作

一、文本格式

代码样例

<h1>一级标题</h1>
<p>这是普通段落，<strong>加粗</strong>、<em>斜体</em>、<mark>高亮</mark>、<small>小号文字</small>。</p>

效果展示

一级标题

这是普通段落，加粗、斜体、高亮、小号文字。

二、装饰方法

代码样例

<p style="color:#f472b6; text-decoration:underline;">
	装饰后的文本
</p>

效果展示

装饰后的文本

提示：也可以通过 CSS 类统一管理颜色、边框、阴影与背景效果。

三、排版方法

代码样例

<div style="display:flex; gap:12px;">
	<div>模块 A</div>
	<div>模块 B</div>
	<div>模块 C</div>
</div>

效果展示

模块 A
模块 B
模块 C

表格排版示例

标签作用常见用途
<header>页面头部标题、导航
<main>主体内容主要学习内容
<section>内容分区模块化组织页面

四、图片、音频等媒体插入

代码样例

<img src="image.jpg" alt="示例图片">
<audio controls>
	<source src="music.mp3" type="audio/mpeg">
</audio>

效果展示

图片展示

说明：音频地址可替换为本地或网络资源。

五、链接跳转插入方法

代码样例

<a href="#text">跳转到文本格式</a>
<a href="https://www.example.com" target="_blank">外部链接</a>

效果展示

跳转到文本格式

跳转到媒体插入

打开外部链接

六、高级操作

代码样例

<details>
	<summary>点击查看更多内容</summary>
	<p>可用于折叠说明、FAQ、课程笔记等。</p>
</details>

效果展示

点击查看更多内容

可用于折叠说明、FAQ、课程笔记等。

课程学习建议

先学文本与链接，再学布局与媒体，最后练习表单、语义标签和页面结构设计。
