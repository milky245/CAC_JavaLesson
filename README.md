# CAC_JavaLesson
Java lesson from Tokyo IT School. For CAC Corp freshers training.

## Repository Structure

This repository is divided into three main parts:

### 1. ClassNotes
Contains class notes for each lesson.

### 2. SourceCode
Contains the source code for each class, organized by lesson.

- `SourceCode/lesson1/` – Source code for Lesson 1

### 3. 100 Question
Contains the 100 Java Quiz from IT school, including official answer and my answer.
